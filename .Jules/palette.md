## 2026-03-27 - Repository is Headless CLI
**Learning:** This repository (Particelle) is a headless Rust CLI application with no web UI, HTML, DOM, or graphical components. Standard web UX/a11y improvements (like ARIA labels, CSS, etc.) are inapplicable here.
**Action:** Stop and do not create a PR when looking for web UI UX enhancements in this repository.
